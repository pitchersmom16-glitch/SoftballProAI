# SoftballProAI

[![CI](https://github.com/pitchersmom16-glitch/SoftballProAI/actions/workflows/ci.yml/badge.svg)](https://github.com/pitchersmom16-glitch/SoftballProAI/actions)

Industry-standard TypeScript monorepo for AI-powered fastpitch coaching.

## Features
- React + Vite frontend
- Node.js + Express backend
- Stripe integration
- Parent-first onboarding flow
- Automated tests, linting, formatting, and CI

## Scripts
- `npm run dev` — Start development server
- `npm run lint` — Lint code
- `npm run format` — Format code
- `npm run check` — Type-check
- `npm test` — Run tests
